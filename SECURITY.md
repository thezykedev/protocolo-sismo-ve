# Security policy

## Supported versions

Solo la rama `main` recibe parches de seguridad y corrección de datos urgentes.

## Reporting a vulnerability

Si encuentras un problema que pueda comprometer la seguridad o la confiabilidad de la información:

1. No abras un `issue` público.
2. Envía un correo a través de [github.com/thezykedev](https://github.com/thezykedev) con los detalles y los pasos para reproducir.
3. Espera confirmación antes de divulgar públicamente.

## Scope

Datos verídicos y oportunos pueden salvar vidas. Cualquier cambio que afecte contactos, filiales, protocolos o instalación offline requiere verificación con fuente oficial antes de fusionarse.
